# Command Line Interface

`qpeek` is designed to be fully controllable from your terminal.

## Usage

```bash
qpeek [OPTIONS] [FILE]
```

## Commands (Draft)

- `qpeek my_cheatsheet.md` - opens the specific markdown file in the viewer.
- `qpeek --daemon` - starts the background daemon manually.
- `qpeek --kill` - gracefully shuts down the background daemon.
- `qpeek --version` - prints the current version.
