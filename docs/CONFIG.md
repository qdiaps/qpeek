# Configuration
`qpeek` is zero-config by default, but highly customizable. The configuration file is located at `~/.config/qpeek/config.json`.

## Default Configuration
```json
{
  "theme": "system",
  "window": {
    "width": 800,
    "height": 600
  },
  "parser": {
    "strict_mode": true
  }
}
```

_(Note: Config structure is subject to change as the project evolves)_
